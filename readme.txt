This file describes the brief functions of each M-file

- bs2mat.m, mat2bs.m
Convert bitstring to coding-matrix and vice versa

- codegen.m
Generate OVA, AP, RHC, RAN coding-matrix

- codeval1.m
Classifiers benchmarking for OVA and AP

- codeval2.m
Classifiers benchmarking for ECOC

- consist.m, svtol.m, mysvc.m, mysvcfwd.m
Related with Support Vector Machine training and testing

- csroutput.m
Outputs given classifier, training data, label, and testing data

- csrtrain.m
alpha, bias (and linear weights) given classifer, training data, label

- cvpar.m
Data partition for CV

- datanorm.m
Data normalization to have zero mean and unit standard deviation

- mulmfcn.m, diversfcn.m, hammfcn.m
Fitness functions for MLM, DIV, HAM

- ecoc.m, ecoctrain.m, ecocfwd.m
ECOC ensemble classifier setup, training and testing

- ecoct.m, ecocv.m
Outputs of ECOC testing and CV given training and testing data

- gacodegen.m
Test GA using different parameters

- innrprod.m, lossfun.m
INNR and LOSS decoding functions

- kf.m
Kernel function

- myplatt.m
Infer sigmoid parameters by n-fold CV

- plr.m, plrtrain.m
Penalized logistic regression setup and training

- ranpearson.m
Calculate Pearson correlations of errors and matrix measures

- s2nrank.m
Rank genes by signal-to-noise ratio

- sigmoid.m
Given sigmoid parameters, convert outputs to probabilities

- truecode.m
Generate MLM, DIV, HAM coding-matrices
